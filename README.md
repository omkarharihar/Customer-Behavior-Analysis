# Customer Behaviour Analysis

## Overview

This project focuses on analyzing customer behavior data using Python, SQL, and Power BI. The objective is to extract useful insights from raw data through data cleaning, exploratory data analysis (EDA), SQL querying, and dashboard visualization.

---

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* PostgreSQL / MySQL / SQL Server
* Power BI
* Gamma AI

---

## Project Steps

### 1. Data Loading

* Imported dataset using Python and Pandas

### 2. Data Cleaning

* Removed null values and duplicates
* Fixed column names and data types
* Cleaned inconsistent data

### 3. Exploratory Data Analysis (EDA)

* Analyzed customer trends and purchasing behavior
* Created visualizations and charts

### 4. SQL Analysis

* Imported cleaned data into SQL database
* Performed analytical queries using:

  * GROUP BY
  * JOIN
  * COUNT
  * AVG
  * SUM

### 5. Power BI Dashboard

* Built an interactive dashboard with insights and KPIs

### 6. Reporting & Presentation

* Created project report and presentation using Gamma AI and PowerPoint
